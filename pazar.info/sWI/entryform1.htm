PAZAR - Data Submission

Some hints:

- You can use this interface to enter data to an already existing project or to a new one. For the latter, you just have to create a new project below and it will be added in your list of existing projects.

- TF centric submissions are based on a specific Transcription Factor from which you want to annotate binding sites.
CRE centric submissions are annotations of a Cis Regulatory Element (CRE) from which you have evidence of a role in gene regulation (binding or other).
